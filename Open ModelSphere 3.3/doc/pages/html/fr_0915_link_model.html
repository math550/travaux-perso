Le modèle de liens


Le modèle de liens contient des objets de lien. Puisqu’on lui donne un nom, le modèle de liens peut avoir une signification sémantique (c'est-à-dire Système A - Système B correspondance des liens).

Un lien est un mécanisme extrêmement flexible qui représente un rapport de type 'plusieurs à plusieurs' entre des objets sémantiques. Le rapport est directionnel, donc le lien peut avoir des objets source et/ou cibles.

Les modèles de liens sont employés pour avoir une certaine correspondance entre les différents types d'objet.

Pour créer ce modèle, cliquez avec le bouton droit sur Projet, puis, du menu contextuel, Ajouter > Modèle de liens.


Pour lier des objets ensemble, vous pouvez choisir entre deux possibilités. Vous pouvez :
  1. Cliquez avec le bouton droit sur Modèle de liens, et choisissez Ajouter > Lien dans le menu contextuel.

  2. Cliquez avec le bouton droit sur le lien et choisissez Propriétés.

  3. Choisissez si c'est un objet source ou cible.


Cependant, il y a une façon rapide de procéder :
  1. Choisissez deux objets. (Exemple : Table 1 et Table 2)

  2. Repérez l'outil de création de lien dans la barre d'outils. Sélectionnez-le.

  3. Choisissez le modèle de liens.

  4. Choisissez lequel sera l’objet source.

  5. Finalement, le lien est établi entre la Table 1 et la Table 2.
