Adding resources


A resource can be linked to processes. When it is, the resource’s graphic representation is displayed within the linked process. It displays the name and/or time and/or cost of each resource of the model that is linked to the process.

To add a resource:
  1. From the explorer, right-click the process model.

  2. From the pop-up menu choose Add > Resource.

To assign a resource to a process:
  1. From the explorer, choose a ressource by clicking on it.

  2. Hold the mouse button and drag the ressourse over a process either in the diagram window or the explorer.

  3. Drop the ressource by releasing the mouse button when the process becomes highlighted and the icon underneath the cursor becomes '+'.
OR
  1. In the main menu, choose Display > Toolbars > resources. The resources tool will appear.

  2. In the resources tool, choose one or many resources to assign. Select one of the three arrows, depending if you plan to add, put identical or delete your selection to the targeted object.

  3. Click on the targeted object.

The Resources list window contains a menu that gives access to pages linked to each resource. After you select the name of a resource, you can access the pages associated with it by choosing the title of the desired page from the menu.


Properties

The Design tab of the Properties window provides the following information:
  • the resource name

  • its numerical identifier

  • its alias, that you can use to display another name

  • the name of a category if you want to group resources

  • its physical name

  • its user, for example if the resource is a machine

  • a cost and cost time unit

  • and a description.

The window also displays the resource work load and work load time unit. The work load is the sum of all the usage rates entered for the processes where the resource is used.
