La recherche d'objets


La fonction Rechercher permet de trouver des objets dans l'Explorateur selon une multitude de critères et d’options.

Le mécanisme de recherche est basé sur les expressions régulières (grep voir ci-dessous pour l'explication). Aussi, les objets, de même que la portée et les options sélectionnés que vous avez précédemment recherchés seront gardés en mémoire pendant la session actuelle.

Pour employer cette fonctionnalité, choisissez Edition > Rechercher depuis le menu principal. Vous pouvez également cliquer sur l'outil Rechercher situé dans la barre d'outils Édition. La fenêtre suivante apparaîtra.




Les options
  • Respect de la casse : compare la casse exacte de la chaîne d'objet. (Exemple :MéLaNge dEs DeUX CaSse)

  • Mots complets seulement : compare la chaîne exacte seulement.

  • Expressions régulières (grep) : inclut des caractères de remplacement grep.

  • Récursif : recherche dans chaque modèle sous la sélection, incluant les sous-modèles de données, de façon récursive.

Note : une expression régulière est une chaîne de caractères où on donne une signification spéciale à certains caractères en ce qui concerne l'appariement. grep permet à l'argument de chaîne d'être considéré comme une expression régulière, qui est une façon de spécifier une chaîne permettant à certains métacaractères (caractères de clavier spéciaux comme le point) de remplacer d'autres caractères ou définir la façon dont l'appariement doit procéder.


La portée
  • Nom : compare l'objet à rechercher avec le nom de l'objet.

  • Nom physique : compare l'objet à rechercher avec le nom physique de l'objet.

  • Alias : compare l'objet à rechercher avec l’alias de l'objet.

  • Description : compare l'objet à rechercher avec la description de l'objet.

  • Tout autre champs de texte : compare l'objet à rechercher avec les attributs de chaîne de tous les autres objets incluant les propriétés multi-lignes (c'est-à-dire le corps de la procédure).

Si vous voulez réduire l’étendu de votre recherche, faites votre sélection au niveau du modèle au lieu du projet.
