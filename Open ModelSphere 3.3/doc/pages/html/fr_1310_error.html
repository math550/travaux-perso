En cas d'erreur


Dans le cas ou une opération est avortée, une fenêtre d'erreur apparaît explicant la source du problème. Appuyez sur OK afin de poursuivre. Il est alors recommandé de sauvegarder votre modèle et de reprendre l'opération. Si l'erreur se reproduit de nouveau, nous vous conseillons de quitter et de redémarrer l'application avant une nouvelle tentative. Si l'erreur est toujours là, vous pouvez tenter de retirer de votre modèle les objets suceptibles d'être la source du problème.
