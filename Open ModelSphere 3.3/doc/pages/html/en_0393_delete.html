Deleting Objects and Graphical Representations


Depending on the sort of object you operate (semantic / graphic), you will have the option to delete an object (Delete from Model) or only the graphical representation (delete). These two options are available in the pop-up menu of any object selection.

Delete from Model:
This action will delete the semantic object. (Including the deletion of all it graphicals representation in all diagrams of the project.)

Delete:
This action will delete only the graphical representation. The semantic object will still exist and other graphical representations too.
