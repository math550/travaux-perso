Utiliser les notes graphiques

Une note graphique sert à ajouter un commentaire sous forme graphique dans le diagramme. Ces notes peuvent être liées graphiquement aux objets graphiques qu'elles décrivent.

Contrairement aux champs Description des éléments du modèle, ces notes ne sont pas utilisées lors de la génération de code.

Pour ajouter une note graphique au diagramme :
 

1. Choisir l'outil de note dans la barre d'outils .

2.  Si la note est reliée à un objet du diagramme, utiliser le lien graphique entre la note et la représentation graphique de l'objet à décrire :
