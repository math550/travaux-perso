Insérer une image


Vous pouvez insérer des fichiers image JPEG ou GIF dans votre diagramme. Cliquez simplement avec le bouton droit dans la fenêtre du diagramme et choisissez Insérer une image du menu contextuel. Une deuxième fenêtre s'ouvrira; vous devez indiquer l’endroit où l'image est placée et finalement choisir l'image.
