Compilation unit concept


Compilation units are resulting Java files generated from the model’s information.


Properties

  • Name: name of the generated java file.

  • Alias: identifies a concept other than by its name (optional).

  • Is compiled: set by the reverse engineering to indicate that the compilation unit was create from a class file.

  • Description: enter a textual description for the compilation unit.


Actions

  • Generate Missing Compilation Unit(s): For a class model or a package, creates a compilation unit for each classes, if they do not already have one.

  • Add > Compilation Unit: For a class, creates a compilation unit if it does not already have one.
