Utiliser la grille
 

Pour bien aligner les représentations graphiques d'un diagramme, Open ModelSphere permet d'afficher une grille en arrière-plan du diagramme.

La taille du grillage est définie par l'utilisateur.  La grille peut être désactivée si non-requis.

Pour définir une grille :
