Utilisation des styles


Utiliser les styles pour modifier l’apparence du projet

Les styles font référence à un jeu d'options graphiques organisés par sorte de diagrammes. Plusieurs styles différents peuvent être utilisés pour chaque type de diagramme pour ajouter de l'information et une touche plus personnelle à vos projets. Format > Styles du projet > Onglet

Un diagramme a un style par défaut qui lui est assigné afin de déterminer le style qu'emprunteront les nouvelles représentations graphiques qui y seront créées. Vous pouvez changer ce style par défaut via le menu :Format > Style du diagramme par défaut.


Éditer les styles

Les différents styles sont spécifiques à un type de modèle. Toutefois, il est possible de créer un nouveau style ou bien d'en importer un à l’intérieur du projet en cours ou dans un projet différent.



Caractéristiques
  • Affichage de la structure : sélection des attributs à afficher.
  • Affichage du descripteur : affiche le nom de l’objet / nom physique/ alias.
  • Formatage du descripteur : options de formatage de colonnes spéciales.
  • Police de caractères : options de formatage de la police de caractères des descripteurs affichés.
  • Préfixe : texte ou préfixe d’image de table, de vue et de colonne.
  • Couleur : couleur pour le remplissage, le trait et le texte.
  • Style du trait : tiret ou mis en évidence.
  • Affichage basé sur le modifiant : contrôle l’affichage du champ de classe Java basé sur l’attribut de ce champ.
