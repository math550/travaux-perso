Export / Import XML


Exporter un projet en format XML

En plus de pouvoir sauvegarder ses projets dans un format de fichier binaire (.sms), ModelSphere peut les sauvegarder dans le format XML.

Pour ce faire, choisissez Exporter vers > Format XML... dans le menu Fichier.


Importer un fichier XML

Les modèles exportés par la fonctionnalité d'export XML de ModelSphere, peuvent être ré-importé.

Pour ce faire, choisissez Importer de > format XML... dans le menu Fichier.

NB: Seuls les fichiers XML exportés de ModelSphere peuvent être importés. Si vous essayez d'importer d'autres sortes de fichier XML, ce processus sera un échec.


Le pour et le contre du format XML en comparaison avec le format binaire (.sms)

Le format XML rend aussi possible une meilleur conservation des modèles dans des outils de gestion de fichiers. Parce que le format XML, étant un format texte, laisse la possibilité à ces outils de montrer les différences entre les différentes versions archivées. Ce qui n'est pas possible avec le format binaire (Seul ModelSphere est capable de lire et de montrer les différences entre deux modèles binaire. Voir l'intégration).

SMS (format binaire)
format XML
POUR : 
  • génère des fichiers plus petits
  • était supporté dans la version précédente de ModelSphere
CONTRE : 
  • ne peut pas être édité avec un éditeur texte
  • correction des fichiers corrompus pratiquement impossible
  • ne peut pas être décripté par opérations sur des chaînes de caractères
POUR :
  • peut être édité avec un simple éditeur de texte
  • les fichiers corrompus peuvent être corrigés
  • peut être décripté par un interpréteur XML
CONTRE :
  • génère de plus gros fichiers
  • seulement dans ModelSphere 2.1 et plus récent



Lire les fichiers XML dans d'autres logiciels

Parce que ce format .xml est spécifique à ModelSphere, il n'est pas possible d'importer les fichiers .xml de ModelSphere dans d'autres outils qui n'ont pas été conçus à cet effet. Néanmoins, il est possible de décripté les fichiers .xml de ModelSphere avec un interpréteur XML standard et donc d'écrire un programme pouvant lire ce format xml.
