Les diagrammes


Selon le type de modèle que vous utilisez dans ModelSphere, vous pouvez ajoutez un ou plusieurs diagrammes correspondant, qui vous permettront la création et l'élaboration graphique de votre modèle.

Diagramme de données relationnelles
Diagramme de processus
Diagramme de classes
Diagramme de domaines
Diagramme d'items communs


Pour ajouter un diagramme à votre modèle:
  1. Sélectionnez l'icone du modèle dans l'explorateur.

  2. Cliquez-droit sur l'icone et choisissez Ajouter > Diagramme. Ou cliquez sur Ajouter dans la barre d'outil et choisissez Diagramme.



NB:
Les diagrammes de processus sont disponibles uniquement sous le contexte et sous les processus. (Pas directement sous le modèle).
Le diagramme de classes quant à lui, est aussi disponible sous les paquetages.
Il est possible de rendre la création des diagrammes automatique à la création des modèles. Voir Modification des options.



Diagramme vide...

Si vous ajoutez un diagramme, celui-ci apparaîtra vide même si des objets sont déjà existants dans le modèle. C'est que les représentation graphiques des objets n'existent pas encore, car elle sont indépendantes. Pour les faire apparaître, choisissez Ajouter les objets du diagramme dans le menu contextuel du diagramme.
