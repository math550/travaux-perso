Le zoom, la vue d'ensemble et la loupe


Les outils d’observation

Il y a trois outils disponibles :
  1. le zoom
  2. la vue d’ensemble
  3. la loupe.
Le zoom

Votre modèle devenant de plus en plus grand, vous aurez tantôt besoin d’une vue d’ensemble, tantôt d’une vue agrandie pour travailler plus aisément. Une façon rapide d’obtenir ces vues est de changer le pourcentage du zoom. Dans la barre d’outils Affichage, utilisez la flèche à côté du pourcentage de graduation et choisissez un nouveau pourcentage.



Vous pouvez aussi changer le pourcentage en vous servant des boutons zoom avant et zoom arrière .


Les fenêtres vue d'ensemble et loupe

Parfois votre modèle deviendra si grand qu'il sera difficile d'y travailler. Pour contrer cette difficulté, Open ModelSphere offre une navigation adéquate et facile pour de grands diagrammes. Nous vous recommandons de combiner les trois fenêtres suivantes sur votre écran : la vue d'ensemble, la loupe et le diagramme. Pour montrer ces fenêtres, dans la barre d'outils Affichage, choisissez les outils Montrer la vue d'ensemble et Montrer la loupe .
Les commandes pour montrer ces trois fenêtres sont aussi disponibles dans le menu Affichage
