Target system


Target systems come with data types. You choose the target system regarding the DBMS (DataBase Management System) used for your data model's development. They are defined in a file and added to the project. It must be noted that although there is only one user node per project, there can be many target systems.
The default target system in Open ModelSphere is called Logical (Logical 1.0).


Setting the target system

When creating a data model, Open ModelSphere requires a target system.
To choose your target systems, select Tools > Setup Target system.







NOTE: A target system cannot be removed from a project if it is used by an object.
