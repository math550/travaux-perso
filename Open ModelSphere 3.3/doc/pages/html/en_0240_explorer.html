Using the explorer


Situated on the left side of your screen, the object explorer is an intuitive and hierarchical navigation tool that allows users to see and manipulate the components of relational, process or object-oriented models in Open ModelSphere. Object explorer groups components into projects which include built-in and user node. Open ModelSphere’s explorer works very much like Windows explorer. Information is stored in a hierarchical manner in nodes that can be expanded or collapsed, providing a horizontal or vertical split view. In addition to its browsing capabilities, you can use the explorer to add, modify, copy or delete information (objects) with a simple mouse selection.
For an easier navigation, the explorer can be set to a top-bottom view, a left-right view or a single view . It can also be hidden, thus allowing more space for the diagram window.
