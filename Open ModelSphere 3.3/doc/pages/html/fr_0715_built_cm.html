Créer un modèle de classes


Vous pouvez créer des composantes d’application Java, soit en réutilisant des composantes existantes, ou en ajoutant de nouvelles composantes dans un modèle de classes. Dans le dernier cas, vous pouvez créer le modèle de deux façons différentes : (1) dans l’Explorateur ou (2) dans la fenêtre du diagramme.

D'abord, vous devez cliquer projet avec le bouton droit et, dans le menu contextuel, choisir Ajouter > Modèle de classes.



Vous pouvez accepter le nom de modèle par défaut, modèle de classes, ou vous pouvez immédiatement assigner un nom plus significatif au modèle.

Vous pouvez également créer des composantes d’application Java en réutilisant des composantes existantes. Par exemple, la fonction copier/coller peut être utilisée sur des objets compatibles, qu’ils soient à l’intérieur ou à l'extérieur du projet. D'autre part, il y a la fonction de déplacement qui peut être appliquée aux objets compatibles s’ils sont dans le même projet (sur le menu contextuel de l'objet, choisissez Déplacer).
