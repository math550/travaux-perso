La fenêtre de listes


La fenêtre de listes d’objets est une alternative à la fenêtre Explorateur. Elle vous permet de voir l'information et de naviguer parmi les données. Facile à utiliser, la fenêtre de listes est pratique et avantageuse. Voici les nombreuses options offertes :
  1. Utilisez le glisser-déposer pour changer l'ordre des colonnes.

  2. Affichez ou cachez n'importe quelle colonne : bien que, vous ne puissiez pas supprimer ou cacher la dernière colonne. La liste de colonnes par défaut (champs) peut être définie pour une liste spécifique. Si aucun choix n'est fait, tous les champs (domaines) possibles seront montrés. Les colonnes peuvent être ajoutées via un menu contextuel, que vous trouverez en plus de la table.

  3. Triez les colonnes en ordre croissant ou décroissant; cliquez simplement sur le nom de la colonne. L'ordre utilisé sera indiqué par un triangle tête vers le haut et un dégradé de couleur pâle à foncé pour l'ordre croissant (l’inverse pour l’ordre décroissant).

  4. Redimensionnez la longueur des colonnes.

  5. Sauvegardez les différentes configurations séparément.

  6. Configurez, via le menu contextuel, l'exposition horizontale ou verticale de lignes.

  7. Accès aux menus contextuels en cliquant avec le bouton droit sur un objet dans la liste.


Vous pouvez afficher la fenêtre de listes de trois façons :
  1. Dans le menu principal choisissez Affichage > Liste > le champ désiré.

  2. Dans la barre d'outils Navigation, choisissez l'outil Liste et le champ désiré.

  3. De l'explorateur ou la fenêtre du diagramme, choisissez l'objet dont vous voulez voir la liste en le cliquant avec le bouton droit et du menu contextuel choisissez Liste.

Les listes sont très pratiques puisqu’elles permettent de naviguer et d’observer sans effort. Si vous désirez éditer un objet :
  • Choisissez l’objet de la liste et faites vos modifications dans le panneau de conception.

  • Cliquez l’objet avec le bouton droit et, du menu contextuel, choisissez Propriétés (ou cliquez sur l'outil Propriétés dans la barre d'outils Navigation).


Les listes sont extrêmement utiles pour naviguer et afficher les nombreux objets de votre projet.
