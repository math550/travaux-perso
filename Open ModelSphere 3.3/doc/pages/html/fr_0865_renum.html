Renuméroter


La commande Renuméroter permet de renuméroter la totalité des représentations graphiques d’un même genre dans un diagramme. Elle fait apparaître la fenêtre Renumérotation qui offre des options pour la Renumérotation sélective.

Pour choisir les paramètres :
  1. Dans le menu Outils, choisissez Paramètres de renumérotation.

  2. Indiquez quel est le premier numéro à utiliser pour effectuer la renumérotation sélective dans la case Renuméroter à partir de.

  3. Précisez l’intervalle de la renumérotation dans la case Incrémenter de.

  4. Cliquez sur Fermer.



Pour effectuer une renumérotation sélective :
  1. Double-cliquez sur l’outil Renumérotation de la barre d’outils Création de processus.

  2. Cliquez chaque représentation séparément pour les renuméroter.

  3. Double-cliquez l’outil Sélection.
