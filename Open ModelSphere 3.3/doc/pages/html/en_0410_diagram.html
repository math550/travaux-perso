Customizing the drawing area


The drawing area can be set, allowing the user to select the number of pages available for the diagram. It is also possible to modify the printer scaling. To use this feature:
  1. Right-click a blank space of the diagram window.
  2. Choose Set drawing area from the pop-up menu.
  3. For drawing area, hold the left button of the mouse and drag to the desired size.
  4. For printer scaling, move the cursor or enter the number directly.
