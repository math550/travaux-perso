Adding processes


To add a process...
  • From the Explorer window: right-click the Context or a process. Then, from the pop-up menu, choose Add > Process.

  • From the Edit toolbar: click the Add tool and choose Process.

  • From the diagram window: choose the process creation tool and click within the diagram window.

The process is added in a hierarchic manner in the explorer window. The first Process added is attributed the number 1, the second is attributed the number 2, and so on...



The process properties window contains a menu that gives information regarding the processes and linked objects.


Properties

The Properties window provides the following information:
  • the process name.
  • its numeric identifier, which is automatically given when you create a new process but can be modified directly in this window or by renumbering.
  • an alphanumeric identifier that lets you create a specific prefix for the process regardless of its hierarchical level (these two identifiers let you know the relative location of a process in a process tree).
  • its alias, that you can use to display another name.
  • its physical name.
  • a synchronization rule that lets you specify the conditions or events that can trigger the process.
  • a component set used by the process.
  • specification if the process is external to the system being described, and if it is a control process.
  • the costs linked to this process through its children and resources and if the process is exploded or not.
