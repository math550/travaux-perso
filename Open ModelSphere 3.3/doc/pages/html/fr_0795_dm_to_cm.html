Génération d'un modèle de données à partir d'un paquetage


Afin d'accomplir cette opération, vous devez avoir un paquetage ou un modèle de classe. Si vous souhaitez créer des liens, vous devrez également créer un modèle de liens.
  1. Avec le bouton droit, choisissez le modèle de classes ou le paquetage et choisissez Générer un modèle de données.



  2. Choisissez le système cible qui sera appliqué au modèle de données. Vous devez préalablement avoir assigné un système cible à votre projet.

  3. Vous devez également choisir l'emplacement du modèle de données : dans le projet ou dans le paquetage utilisateur. En cliquant la boîte, un choix apparaîtra.

  4. Vous pouvez créer des liens.
    Rappelez-vous, un modèle de liens est exigé si l’option est choisie.
    • Classe (source) à table (cible).
    • Classe (source) au domaine (cible).
    • Champ (source) à colonne (cible).


  5. Cliquez sur le bouton Générer.



Les objets générés

Modèle de données contenant :
  • une table pour chaque classe (les interfaces ne sont pas transférées);
  • un champ d'association et un champ statique ne sont pas transférés en tant que colonne;
  • une association est transférée sur une base continue (une à une);
  • l'héritage est transférée comme les associations; les sous-tables auront une clé de dépendance sur les super-tables.

Modèle de domaines contenant :
  • un domaine pour chaque classe utilisée comme type de champ.

Objets liés :
  • Si l'option de relation est choisie, les objets liens sont générés dans le modèle de liens déterminé.
