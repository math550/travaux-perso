Scinder


La commande Scinder facilite le travail d’équipe. Elle offre la possibilité de scinder un modèle ou une de ses parties. Le résultat de l’opération de scission est un autre fichier sans lien avec l’original.

Avant qu’une scission puisse être exécutée, le modèle doit être ouvert et un processus éclaté doit être graphiquement sélectionné. Cette action scindera le processus du modèle et de ses descendants.

Pour effectuer l'opération de scission, choisissez Scinder du menu Outils.

Après l’opération, un nouveau modèle est créé et reste ouvert. Le processus antérieurement sélectionné devient le diagramme contexte de ce nouveau modèle.

Quand un processus a été scindé d'un modèle, son exemplaire dans le modèle d'origine est automatiquement configuré 'externe'. Un 'E' apparait alors dans son identifiant.
