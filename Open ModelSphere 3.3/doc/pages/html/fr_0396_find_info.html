Trouver l'information


Trouver l'information sémantique

Il est possible de trouver de l'information sémantique quant aux objets graphiques dans la fenêtre du diagramme. Cliquez avec le bouton droit sur l'objet et dans le menu contextuel sélectionnez Rechercher dans l’explorateur. L'objet choisi sera mis en évidence.



Trouver l'information graphique

Il est aussi possible de trouver de l'information graphique quant aux éléments sémantiques dans la fenêtre Explorateur. Cliquez avec le bouton droit sur le mot et dans le menu contextuel sélectionnez Rechercher dans le diagramme.
