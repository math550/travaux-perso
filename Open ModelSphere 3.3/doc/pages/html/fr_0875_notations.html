Formalismes


Lorsque vous créez un nouveau fichier, le formalisme Gane-Sarson est sélectionné par défaut. Vous pouvez opter pour un autre formalisme avant, pendant ou après la création d’un modèle.

Pour sélectionner un autre formalisme :
  1. Cliquez-droit dans le fond du diagramme.

  2. Choisissez le sous-menu contextuel Formalisme.

  3. Sélectionnez le formalisme désiré pour ce diagramme. L’aspect des objets graphiques changera selon le formalisme choisi.

Il vous est possible de choisir un formalisme par défaut pour un projet, afin de ne pas avoir à refaire ce choix à chaque nouveau diagramme. Dans le menu contextuel du Projet, vous y trouverez Formalisme par défaut (Modèle de processus).

Si vous ne voulez pas utiliser les formalismes proposés, vous pouvez créer un formalisme personnalisé. Voir Les Formalismes.
