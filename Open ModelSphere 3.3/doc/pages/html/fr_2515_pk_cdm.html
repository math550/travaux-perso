Ajouter des clés primaires


Une clé primaire est un attribut (ou combinaison minimale d’attributs) qui identifie de façon unique chaque rangée dans une entité.

Les clés primaires peuvent être créées de deux façons différentes : soit en utilisant la barre d’outils, soit en entrant directement l'information dans la fenêtre de propriétés du modèle.
Exemple : numéro d'employé, code de l’entreprise, numéro du département.
  1. Sélectionnez l'outil Création de clé dans la barre d'outils Création.

  2. Choisissez l’attribut que vous voulez désigner comme clé. Le nom de l’attribut sera souligné pour indiquer que c'est un attribut clé primaire. (Si vous choisissez le mauvais attribut, réutilisez le même outil pour enlever le souligné.)


    3. Rappelez-vous que les clés primaires peuvent être composées de plusieurs attributs. Vous n’avez qu’à choisir les attributs complémentaires qui feront partie de la clé de la même manière.

  3. Ajoutez les clés primaires à votre diagramme tel qu’indiqué dans le modèle de données suivant.



Dans Open ModelSphere, les clés primaires sont soulignées afin de les distinguer des autres attributs. Dans l'exemple ci-dessus, les quatre entitées ont une clé primaire composée d'un seul attribut. Par exemple, dans l'entité Projet, le Numéro du projet d’équipe est utilisé pour identifier chaque projet, et chaque Numéro d'employé identifie de façon unique chaque employé dans l'entité Employé.
