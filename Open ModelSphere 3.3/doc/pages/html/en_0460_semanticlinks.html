Creating semantic and graphical links


A link is a powerful construct to build an association between any semantic objects, e.g. it is possible to link a data store in a process model to a data model or to a relational table. A link may have several source objects and several destination objects as well.

Links are particularly helpful in a tool that combines business process, relational data and object-oriented modeling elements. In case the linked elements have graphical representations in the same diagram the link can be graphically displayed as well.
