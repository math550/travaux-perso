Concepts spécifiques et attributs


Tous systèmes cibles peuvent avoir des concepts et attributs spécifiques. Par exemple, si vous copiez une table d'un modèle de données logique et la collez dans un modèle de données d'Oracle, les propriétés changeront. On ne le montrera pas dans l'Explorateur, cependant, si vous regardez les propriétés de la table (dans la fenêtre de propriétés), vous remarquerez qu'elles ont été converties. Observez le tableau de la page suivante.
