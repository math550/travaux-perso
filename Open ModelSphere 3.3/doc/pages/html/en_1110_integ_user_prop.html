Integrating the user properties


Once you have established your user properties, you will find them in the object's properties window under the Design tab or in the design panel. User properties are displayed in italics in the design tab, as shown in the picture below.



In the user properties window, you can check the properties, under the Design tab.



It is also possible to verify which properties have written values and what are those values. Selecting each user property separately, look under the Used by tab.
