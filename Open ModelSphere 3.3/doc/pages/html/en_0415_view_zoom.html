Zoom, overview and magnifier


Viewing tools

These are the three available tools:
  1. Zoom
  2. Overview window
  3. Magnifier window
Zoom

As your model becomes larger, you might want to get an overview of it and then an enlarged view for working purposes. A quick way to do this is to change the zooming percentage. In the display toolbar, select the arrow next to the current scaling percentage and choose a new one.



You can also change the percentage by click on the zoom-in an zoom-out buttons.


Overview and magnifier windows

Sometimes your model becomes so large that it is difficult to work on it. Open ModelSphere offers adequate and easy navigation for large diagrams. We recommend combining all three following windows on your screen: the overview window, the magnifier window as well as the diagram window. To display these windows, select the Show Overview Window tool or Show Magnifier Window tool in the Display toolbar.
The commands for displaying all three windows are also available in the Display menu.
