Ajouter une entité externe


La représentation graphique d’une entité externe affiche le nom, l’alias, le nom physique et l’identifiant de chaque entité externe du modèle. Vous pouvez aussi passer d’une entité externe standard à une entité externe de contrôle (affichée en pointillé dans la fenêtre graphique) en cochant la case Contrôle dans la fenêtre de propriétés.

Pour ajouter une entité externe :
  1. Sélectionnez l’outil Création d’entités externes dans la barre d’outils Création de processus.

  2. Cliquez un espace blanc dans la fenêtre du diagramme.

La fenêtre de propriétés d’un flux contient un menu qui fournit de l’information concernant les flux et les objets qui leurs sont rattachés.



Les propriétés

L’onglet Conception de la fenêtre de propriétés fournit l’information suivante :
  • le nom de l’entité externe

  • l’identifiant numérique qui lui est attribué automatiquement lors de sa création (mais qui peut être modifié directement dans cette fenêtre ou par renumérotation)

  • l’alias que vous pouvez utiliser pour afficher un autre nom

  • le nom physique

  • une définition

  • la précision si c’est une entité de contrôle.

  • une courte description
