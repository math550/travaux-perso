Les duplicatas


Open ModelSphere vous permet de créer un duplicata d'objet graphique à partir du menu contextuel :
  • d'un objet sémantique (sélection dans l’explorateur).
  • d'un objet graphique (sélection dans le diagramme).
La cible est toujours un diagramme et ce dernier devra être ouvert quand les nouveaux objets graphiques seront créés au centre de la fenêtre.


Les limitations suivantes s’appliquent à la liste de diagrammes ciblés disponibles:

Diagrammes de classes :
  • Tout diagramme de classe Java du projet.
  • Tout diagramme de paquetage Java du projet.
Autres diagrammes :
  • Tout diagramme du même modèle est accepté.
