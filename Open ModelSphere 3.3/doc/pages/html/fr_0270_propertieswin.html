La fenêtre de proriétés


Chaque objet sémantique a une fenêtre de propriétés. Cette fenêtre a plusieurs options, comme modifier l'information d'un objet, relier l’objet à d'autres objets du référentiel et afficher, ajouter, modifier ou supprimer des objets composites. Dans une fenêtre de propriétés, l'information peut être affichée sous forme de description ou de liste. Si c'est une liste, vous pouvez sélectionner les objets et les menus contextuels seront disponibles.

Pour afficher la fenêtre de propriétés d'un objet :

1. Choisissez l'objet pour lequel vous voulez voir la fenêtre de propriétés.

2. Cliquez avec le bouton droit et, du menu contextuel, choisissez Propriétés.

Ou : dans la barre d'outils Navigation, cliquez sur l'outil Propriétés =>




- Pour changer la valeur d'une propriété, double-cliquez dans la section blanche.

- Si vous faites des changements, vous devez cliquer sur Appliquer pour que le changement soit tenu en compte.

- Onglets: La fênetre de propriété vous offre plusieurs panneaux de propriétés que vous pouvez accéder par le biais des onglets.
