Supprimer les objets et supprimer les représentations graphiques


Selon le type d’objet avec lequel vous travaillez (sémantique / graphique), vous aurez accès à l'option pour supprimer un objet (supprimer du modèle) ou pour supprimer sa représentation graphique seulement (supprimer). Ces deux options sont disponibles dans le menu contextuel de n'importe quel objet.

supprimer du modèle :
Cet action supprimera l'objet sémantique. Ce qui entraînera la suppression de toutes ses représentations graphiques utilisées parmi les diagrammes du projet.

Supprimer :
Cet action supprimera la représentation graphique uniquement. L'objet sémantique continuera d'exister ainsi que les autres représentations graphiques de cet objet à travers le projet.
