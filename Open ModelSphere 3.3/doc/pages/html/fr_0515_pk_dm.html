Ajouter des clés primaires


Une clé primaire est une colonne (ou combinaison minimale de colonnes) qui identifie de façon unique chaque rangée dans une table.

Les clés primaires peuvent être créées de deux façons différentes : soit en utilisant la barre d’outils, soit en entrant directement l'information dans la fenêtre de propriétés du modèle.
Exemple : numéro d'employé, code de l’entreprise, numéro du département.
  1. Sélectionnez l'outil Création de clé dans la barre d'outils Création relationnelle.

  2. Choisissez la colonne que vous voulez désigner comme clé. Le nom de la colonne sera souligné pour indiquer que c'est une colonne clé primaire. (Si vous choisissez la mauvaise colonne, réutilisez le même outil pour enlever le souligné.)


    3. Rappelez-vous que les clés primaires peuvent être composées de plusieurs colonnes. Vous n’avez qu’à choisir les colonnes complémentaires qui feront partie de la clé de la même manière.

  3. Ajoutez les clés primaires à votre diagramme tel qu’indiqué dans le modèle de données suivant.



Dans Open ModelSphere, les clés primaires sont soulignées afin de les distinguer des autres colonnes. Dans l'exemple ci-dessus, les quatre tables ont une clé primaire composée d'une seule colonne . Par exemple, dans la table Projet, le Numéro du projet d’équipe est utilisé pour identifier chaque projet, et chaque Numéro d'employé identifie de façon unique chaque employé dans la table 'Employé'.
