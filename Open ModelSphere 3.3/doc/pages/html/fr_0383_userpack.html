Le paquetage utilisateur


Le paquetage utilisateur offre une façon flexible d'organiser l'information d’un projet de façon hiérarchique. Il peut contenir tous les autres types de modèle dans le référentiel, y compris d'autres paquetages utilisateurs. Il peut, par exemple, être utilisé pour représenter une application ou un système. Ce concept est simplement un sous-répertoire du projet.
