Les barres d'outils


Open ModelSphere est doté d’une kyrielle d’outils pour la création ou la modification graphique.
Les barres d’outils regroupent les outils qui sont utilisés dans le processus de conception de modèle. Certaines barres d’outils ont même des fonctions servant à améliorer la présentation graphique des modèles (pour plus de détails sur les outils de dessin et de couleur, voir la section “Surface de travail”). Les autres barres d’outils contiennent les fonctions qui seront utilisées lors de la conception du modèle. Selon le type de fenêtre dans lequel vous travaillez, certains outils ne seront pas disponibles.
Vous trouverez ces outils dans une des barres d’outils suivantes :

Fichier


  • Nouveau
  • Ouvrir...
  • Fermer
  • Enregistrer
  • Imprimer...
Édition


  • Annuler (avec sous-menu de l'historique des modifications)
  • Rétablir (avec sous-menu de l'historique des annulations)
  • Ajouter (avec sous-menu des concepts ajoutables selon la sélection)
  • Copier
  • Coller
  • Supprimer
  • Rechercher...
  • Rechercher le suivant
Affichage


  • Ouvrir l'arborescence (dans l'Explorateur)
  • Fermer l'arborescence (dans l'Explorateur)
  • Rafraîchir
  • Tendre (pour les connecteurs)
  • Mise à l'échelle
  • Zoom avant
  • Zoom arrière
  • Afficher le diagramme
  • Montrer/Cacher la vue d'ensemble
  • Montrer/Cacher la loupe
Format


  • Style (sélecteur)
  • Mettre en évidence (Trait gras)
  • Tiret
  • Couleur du trait (avec sous-menu de couleurs)
  • Couleur de remplissage (avec sous-menu de couleurs)
  • Couleur du texte (avec sous-menu de couleurs)
Navigation


  • Propriétés (selon la sélection)
  • Listes (avec sous-menu des listes disponibles selon la sélection)
Creation

Modélisation de bases de données.


  • Pointeur (outil graphique de sélection et de manipulation)
  • Création de Tables
  • Création de Vues
  • Création d'associations
  • Création de clés

Modélisation de processus.


  • Pointeur (outil graphique de sélection et de manipulation)

  • Création d'entités externes
  • Création de dépôts
  • Création de flux (avec sous-menu pour choisir la forme)
  • Décomposition de processus
  • Renumérotation

Modélisation Objet Java

  • Création de classes
  • Création d'interfaces
  • Création d'exceptions
