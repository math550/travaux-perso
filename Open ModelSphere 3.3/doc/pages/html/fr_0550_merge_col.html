Fusionner les colonnes


Pour fusionner des colonnes, elles doivent être compatibles. Ainsi, la fusion de colonnes est possible quand vous avez ces combinaisons :
  • Base-base : si elles ont le même type de données.
  • Base-étrangère : si elles ont le même type de données.
  • Étrangère-étrangère : si elles ont le même type de données et que les deux colonnes étrangères réfèrent à la même colonne de clés.

Le mécanisme

Les colonnes sont fusionnées, mais les propriétés de colonnes, elles, ne sont pas fusionnées. Donc, les propriétés de la colonne cible sont conservées. Également, tout lien sur l'objet source sera transféré à l'objet cible. Par exemple, si la colonne fait partie d'un index, la colonne cible deviendra la nouvelle colonne d'index. Et pour cette raison, il est seulement possible de séparer les fusions de type étrangères-étrangères.


L'utilisation

La Fusion est employée principalement pour reproduire la clé étrangère et recréer une structure de clé avec l'information existante.
