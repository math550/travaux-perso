La barre d'état


La barre d'état est située au bas de l’écran. Elle donne l'information quant au :
  • message : affichage d'information appropriée de l'application, comme l'ouverture, l'enregistrement d’un projet et des tooltips.
  • projet : affichage du nom du projet actif et son emplacement.
  • base de données : affichage d’informations concernant le RAM et référentiel ainsi que l’icône base de données activé ou désactivé selon le statut de la connexion.
Il est possible de montrer ou cacher la barre d’état.
Pour cela, choisissez le menu Affichage, puis Montrer/Cacher la barre d’état.
