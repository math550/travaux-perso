Générer des diagrammes d'arbres de processus

Les diagrammes de flux de données d'un modèle de processus ne présente que les flux de données entre les processus d'un même niveau.  Pour afficher les liens hiérarchiques entre processus, c'est-à-dire montrer leur décomposition fonctionnelle, Open ModelSphere offre la possibilité de générer un diagramme d'arbres de processus.

Pour générer un tel diagramme,  exploser l'arbre de l'explorateur et sélectionner le diagramme racine de votre modèle.  Cliquer à droite pour obtenir le menu déroulant, et choisir Générer->Diagramme hiérarchique. Un dialogue apparaît alors en donnant les options suivantes :

   
Espacement : normal ou minimal
“Minimal” produira un arbre plus compacte.

Orientation : horizontale ou verticale
“Horizontale” affiche la hiérarchie de processus en rangée de haut vers les bas
“Verticale” affiche la hiérarchie de processus en colonnes de gauche vers la droite.

Alignement : Gauche ou centrer
“Gauche” aligne tous les processus sur la marge de gauche, “Centre” aligne tous les processus justifiés au centre.
“Center” aligns all process rows in the center.
Lorsque l'orientation “Vertical” est active, les options pour l'alignement sont modifiés à “haut ou centre”.

 Ajuster les processus :
Au lieu d'avoir tous les processus de la même taille, cette option permet d'ajuster la taille de chaque processus.
 
Afficher seulement les processus explosés  :
Les noeuds-feuilles ne sont pas affichés dans le diagramme hiérarchique.




Une fois le diagramme hiérarchique générer, il n'est pas permis d'ajouter des processus, acteurs, dépôts et flux.  (La barre d'outils est active, mais l'ajout de nouveaux éléments sera refusé).

Le diagramme hiérarchique est lié à son diagramme original (voir la propriété Diagramme parent dans le panneau de conception); la modification d'un processus dans le diagramme parent sera immédiatement propagée dans le diagramme hiérarchque, mais l'ajout d'un processus dans le diagramme parent ne mettra pas à jour le diagramme hiérarchique.  Pour mettre à jour le diagramme hiérarchique, sélectionner le diagramme hiérarchique dans l'explorateur, et choisir Générer->Diagramme hiérarchique dans le menu déroulant.
