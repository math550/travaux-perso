Ajouter des classes, interfaces, exceptions et paquetages


Vous pouvez ajouter des paquetages, des classes, des interfaces et des exceptions à votre modèle de classes à partir de l’Explorateur.

Les étapes qui suivent vous montrent comment ajouter des paquetages et des classes (interface et exception aussi) à votre modèle.
  1. Cliquez avec le bouton droit sur l'icone du modèle de classes dans l’Explorateur.

  2. Dans le menu contextuel, choisissez Ajouter > Paquetage.

  3. Pour ajouter un paquetage au paquetage, cliquez le premier paquetage avec le bouton droit (dans l’Explorateur) et choisissez de nouveau Ajouter > Paquetage dans le menu contextuel.

  4. Cliquez avec le bouton droit sur le modèle de classes, puis, du menu contextuel, choisissez Ajouter > Classe. Vous pouvez ajouter autant de classes au modèle de classes ou au paquetage que vous désirez.



Vous avez fait une erreur ? Vous pouvez supprimer un objet dans l'Explorateur de ModelSphere. Sélectionnez l'article et cliquez l'icone Supprimer dans la barre d'outils. D’autre part , si vous cliquez l’icone de suppression à plusieurs reprises cela détruira les opérations, une par une.

Si, à tout moment, vous voulez agrandir une hiérarchie dans l'Explorateur, cliquez sur 'la poignée' à côté d'une catégorie.

Si vous préférez créer votre modèle à l’aide de la fenêtre graphique, vous devrez galement Afficher le diagramme.

Habituellement, un changement effectué dans la fenêtre graphique est automatiquement réfléchi dans l'Explorateur de Open ModelSphere.

Si votre modèle requiert la création de plusieurs classes, interfaces, exceptions et/ou paquetages, il est probablement plus simple et plus rapide de créer un diagramme du modèle de classes et ajouter les objets au diagramme.

La section suivante explique comment faire des objets à partir de la fenêtre graphique.


Ajout de classes, interfaces, exceptions et paquetages en utilisant la fenêtre graphique

Pour ajouter une classe, une interface, une exception, ou un paquetage à votre modèle en utilisant la fenêtre graphique (plutôt que l'Explorateur), utilisez l’outil approprié, que vous verrez dans la barre d'outils Création Java : Création de classes , Création d'Interfaces , Création d'exceptions ou Création de paquetages .

Avant d’ajouter des classes aux paquetages, en utilisant la fenêtre graphique, vous devez créer un diagramme pour chaque paquetage.
  1. Cliquez avec le bouton droit sur le paquetage dans l'Explorateur et, du menu contextuel, choisissez Ajouter > Diagramme.

  2. Pour afficher le diagramme, cliquez avec le bouton droit et choisissez Afficher le diagramme à partir du menu contextuel. En utilisant la fenêtre graphique, vous ajouterez maintenant des classes, exceptions et interfaces au paquetage.

  3. Choisissez l'outil Création de classes de la barre d’outils Création Java et cliquez sur un espace blanc de la fenêtre graphique. Une classe apparaîtra.

  4. Vous pouvez faire la même opération avec les autres outils de création pour les exceptions, interfaces et paquetages.
