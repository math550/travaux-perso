La génération de fichiers Java


Pour chacun des concepts de modélisation de classes, il est possible de prévisualiser le code Java qui sera éventuellement généré. Vous pouvez voir le code dans l'onglet Code Java de la fenêtre de propriétés.

Cliquez avec le bouton de droite sur un des concepts d'un modèle de classe et choisissez Propriétés dans le menu contextuel. Cliquez sur l'onglet Code Java.




La génération de fichiers Java (.java)

La fonctionnalité de génération du code Java, génère les fichiers Java à partir des Unités de compilation .
  • Une unité de compilation permet donc de définir le contenu d'un fichier Java.
  • Une unité de compilation peut contenir une ou plusieurs classes.
  • C'est par les unités de compilation que l'on ajoute des importations de classes et paquetages.
On peut lancer la génération des fichiers Java globalement depuis le modèle de classe ou pour une unité de compilation en particulier. Il suffit de choisir Générer > Fichier(s) Java dans le menu contextuel de ces concepts.
Après avoir sélectionné le répertoire de génération, des fichiers java (.java) seront générés pour chacune des unités de compilations.


Générer les unités de compilation manquantes

Quand un modèle prend de l'ampleur, il devient difficile de s'assurer manuellement que tous les classes soient bien liées à une unité de compilation. La fonction Générer > Unités de compilation manquantes, disponible dans le menu contextuel du modèle de classe et des paquetages, permet de générer les unités de compilation pour les concepts qui n'en n'ont pas.
