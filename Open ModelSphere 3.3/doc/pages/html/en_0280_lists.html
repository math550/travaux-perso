Using the lists


The objects’ lists are a useful alternative to the explorer window. They allow you to see information and navigate through the data. Lists are extremely easy to manage. Here are the options offered:
  1. Change the columns’ order by dragging and dropping.

  2. Display or hide any column: although, you can not remove or hide the last column. The default list of columns (also called fields) can be defined for a specific list. If no selection is made, all possible fields will be displayed. Columns can be added via a pop-up menu, which you will find on top of the table.

  3. Sort Ascending or Sort Descending any column; click the column’s name. The sorted order will be indicated by a triangle pointing upward or downward as well as a color gradation (from pale to dark for ascending order).

  4. Resize the columns’ length.

  5. Save each lists’ setting separately.

  6. Set, via the pop-up menu, the horizontal and vertical lines display.

  7. Access pop-up menus by right-clicking an object in the list.

You can display the lists in three ways:
  1. In the main menu select Display > List > the object you want to have listed.

  2. In the toolbar, choose the List tool and the object you want to have listed.

  3. From the explorer or the diagram window, select the object you want to have listed by right-clicking it and from the pop-up menu choose List.

The lists allow numerous viewing and browsing options. And if you wish to edit an object:
  • Select it from the list and easily edit it via the design panel.

  • Right-click this object and choose properties (or click the properties tool in the navigation toolbar).

Lists are extremely useful to display and browse through the numerous objects of your project.
