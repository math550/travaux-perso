Using the properties window


Each semantic object has a properties window. This window has multiple features, namely modifying object information, linking to other repository objects and displaying, adding, modifying or deleting composite objects. In a properties window, information is displayed in a description or a list. If it is a list, object selection is possible and pop-up menus are available.

To see an object’s properties window:

1. Select the object for which you want to see the properties window.

2. Right-click it and from the pop-up menu choose Properties.

Or : in the navigation toolbar, click the properties tool =>




- To change the value of a property, double-click inside the white section.

- If you have made changes, click Apply and Close.

- Tabs: Change tabs to reach other properties of the object.
