Générer de la documentation HTML


Avant de générer un rapport, assurez vous d'avoir choisi le répertoire cible pour des rapports HTML via Outils > Options > Répertoires.

Pour générer un rapport HTML, à partir de l'Explorateur, cliquez avec le bouton droit un modèle. À partir du menu contextuel, sélectionnez Générer > Explorateur HTML.
