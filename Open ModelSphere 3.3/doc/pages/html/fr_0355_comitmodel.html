Le modèle d'items communs

Un item commun est un élément du dictionnaire de données bien défini. Les items communs s'appliquent indépendamment du paradigme choisi, c'est-à-dire pourraient être liés par exemple avec les attributs d'une entité, les colonnes d'une table et/ou les champs d'une classe.

En outre, des items communs sont définis sans égard à tout système-cible.

Il peut avoir une représentation graphique des éléments d'information communs.
