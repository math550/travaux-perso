Adding a system

To add a context to a process model, you must first select the process model in the explorer window...
  • From the main menu: choose Edit > Add > SystemFrom the explorer window: right-click the UMLmodel. Then, from the pop-up menu, choose Add > System
  • From the Edit toolbar: click the Add tool and choose System


To change the “System” name that is automatically assigned by the software, use the description window of the root process. A name change for this process will immediately affect the title bar of its graphic window.
