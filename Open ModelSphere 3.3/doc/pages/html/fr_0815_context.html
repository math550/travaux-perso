Ajouter un contexte


Pour ajouter un contexte à un modèle de processus, vous devez d'abord sélectionner le modèle de processus dans l'explorateur...
  • À partir du menu principal: choisissez Édition > Ajouter > Contexte.

  • Depuis l'explorateur: cliquez-droit sur le modèle de processus et depuis le menu contextuel, choisissez Édition > Ajouter > Contexte.

  • Par la barre d'outils Édition : cliquez sur l'Outil Ajouter et choisissez Contexte.



Vous pouvez changer le nom «Contexte» automatiquement attribué par le logiciel en utilisant la fenêtre de description du processus racine. Un changement de nom se répercutera immédiatement dans la barre de titre de sa fenêtre graphique.
