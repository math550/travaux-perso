Ajouter un index


Un index peut être ajouté automatiquement pour les contraintes primaires, uniques ou étrangères. Quand un index est lié à une clé, Open ModelSphere s'assure que les éléments de colonne d'index sont toujours synchronisés avec la composition de clé correspondante.

Pour générer un index, vous devez sélectionner une des clés (primaire, unique ou étrangère), la cliquer avec le bouton droit et choisir Ajouter un index dans le menu contextuel. Autrement, vous pouvez choisir le descripteur de table > Ajouter > Ajouter un index.

Choisissez ensuite l'index > Propriétés > Éléments indexés > Ajouter (pour ajouter des entrées dans l'index). Vous devrez choisir un espace blanc pour que la fenêtre s’ouvre. Vous pouvez maintenant sélectionner et ajouter vos éléments.
