Convertir un modèle conceptuel de données en modèle relationnel de données

Un modèle conceptuel peut être converti en modèle relationnel. La conversion résout entre autres les relation plusieurs à plusieurs, réassigne les attributs des associations et considère les dépendances de clef.

Il est possible de convertir un modèle conceptuel de données à partir du menu principal, de la fenêtre Explorateur et du diagramme.

  • Du menu principal, choisir Outils > Convertir en Mode Relationnel....
  • De la fenêtre Explorateur, cliquer à droite sur un modèle (ou le diagramme du modèle). Un menu contextuel aparaîtra. De ce dernier, choisir Convertir en Mode Relationnel....
  • Du diagramme, cliquer à droite,choisir Convertir en Mode Relationnel....


 

Le processus de conversion crée un nouveau modèle. Vous pouvez choisir

  • De seulement convertir un modèle dans le diagramme actif ou inclure tous les sous-modèles
  • La notation pour le modèle du cible
  • Absorber les associations optionnelles

Veuillez noter qu’il faut

  • Épurer le modèle
  • Vérifier l’intégrité
avant la conversion.

Un modèle conceptuel et ses modèles convertis sont indépendants les uns des autres, ce qui signifie que les modifications faites après la conversion ne se réflèteront pas dans le modèle converti.
