Intégration des propriétés de l'utilisateur


Dès que vous aurez défini vos propriétés de l'utilisateur, vous pourrez les voir dans la fenêtre de propriétés de l'objet sous l'onglet Conception (ou encore, dans le panneau de conception). Les propriétés de l'utilisateur y sont affichées en italique, tel que démontré dans l'image ci-dessous.



Dans la fenêtre de propriétés de l’utilisateur, vous pouvez vérifier les propriétés, sous l’onglet Conception.



Vous pouvez aussi vérifier à quelle(s) propriété(s) on a attribué une valeur. Pour ce faire, vous devez sélectionner chaque propriété de l'utilisateur séparément. Ensuite, vous verrez l’information affichée sous l'onglet Utilisé par.
