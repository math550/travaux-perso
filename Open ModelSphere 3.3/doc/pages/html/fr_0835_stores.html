Ajouter un dépot


La représentation graphique du dépôt affiche le nom, l’identifiant et l’alias de chacun des dépôts du modèle. Vous pouvez aussi passer d’un dépôt standard à un dépôt de contrôle (dont les contours sont affichés en pointillé dans la fenêtre graphique).

Pour ajouter un dépôt :
  1. Sélectionnez l’outil de création Dépôts dans la barre d’outils Création de processus.

  2. Cliquez un espace blanc dans la fenêtre du diagramme.

La fenêtre de propriétés d’un flux contient un menu qui fournit de l’information concernant les flux et les objets qui leurs sont rattachés.



Les propriétés

L’onglet Conception de la fenêtre de propriétés fournit l’information suivante :
  • le nom du dépôt

  • l’identifiant numérique qui lui est attribué automatiquement lors de sa création (mais qui peut être modifié directement dans cette fenêtre ou par renumérotation)

  • l’alias que vous pouvez utiliser pour afficher un autre nom

  • le nom physique

  • le volume (s’il est connu)

  • la précision s’il s’agit d’un dépôt de contrôle.

  • une description.
