Le concept d'unité de compilation


Les unités de compilation sont les fichiers Java résultants qui sont générés à partir de l'information du modèle.


Les propriétés des unités de compilation

  • Nom : nom du fichier Java généré.

  • Alias : Identifie un concept de façon différente (autre que par son nom).

  • Est compilé : réglé par la rétro-ingénierie afin d'indiquer que l'unité de compilation a été créé à partir d'un fichier classe.

  • Description : permet d'entrer une description en format texte.


Les actions possibles

  • Générer les unités de compilation manquantes : pour les modèles de classes ou les paquetages, crée une unité de compilation pour chaque classe s'il n'y en a pas déjà.

  • Ajouter > Unités de compilation : pour les classes, crée une unité de compilation s'il n'y en a pas déjà.
