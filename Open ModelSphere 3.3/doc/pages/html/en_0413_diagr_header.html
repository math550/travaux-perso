Defining page numbers and titles

It is possible to display the page number on each page of the diagram (see the Displaying the page number and title section). The number can appear at the top or at the bottom of each page.

Instead of displaying page numbers, you can define a title that will be printed on each page.  This title can contain fixed text, such as the name of your organization, but also variable text, such as time, date, or number of pages.  Depending on the settings defined at Displaying the page number and title, the title will appear as a page header (at the top of each page) or a page footer
(at the bottom).

Variable text includes :
 

  • Current page
  • Total pages
  • Current row
  • Total rows
  • Current column
  • Total columns
  • Current time
  • Diagram name
  • Current date
  • Project name

To define the page number and title:
    1. Open the Options dialog by accessing the Tools->Options menu, this will display the dialog:

    2. Define the Format string and press Apply and then Close. In the format string, {1} will be replaced by the current row and {2} by the current column. If your diagram uses five pages horizontally and three pages vertically, your pages will be numbered :

  • 1,1    1,2    1,3    1,4    1,5   for the first row
  • 2,1    2,2    2,3    2,4    2,5   for the second row
  • 3,1    3,2    3,3    3,4    3,5   for the third row
If the format string is defined as "Page {0} of {5}", pages will be numbered Page 1 of 15, Page 2 of 15, and so forth.

If the format string is defined as "Page {0} of {5} - {8} - Modified on {4} at {3}", pages will be entitled "Page 1 of 15 - Main Diagram - Modified on Jul 21, 2004 at 10:35:17 AM" when "Main Diagram" is the name of the diagram, {4} represents the current date and {3} the current time.
