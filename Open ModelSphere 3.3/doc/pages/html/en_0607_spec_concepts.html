Specific concepts and attributes


Any target system can have specific concepts and attributes. For example, if you copy a table from a logical data model and paste it in an Oracle data model, the properties will change. This will not be shown in the explorer window, however, if you look at the table's properties (in the properties window), you will notice that they have been converted.
