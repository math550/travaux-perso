Le modèle de données


Il s'agit d'un modèle relationnel basé sur un graphique fait de tables, colonnes, associations binaires et quelques extensions d'objet. Un modèle de données est employé comme véhicule de communication pour construire et décrire une bases de données. Ce concept contiendra toute l'information relationnelle des différents contenants des données d'une base.
Pour une modélisation efficace, le modèles de données sera rattaché aux autres composantes de modélisation de données par le biais de la Base de données .
