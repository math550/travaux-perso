Creating user properties


User properties are extended attributes, defined by the user, of existing project concepts, in order to customize their description. Each concept may have multiple user properties.

If you wish to create your own defining properties for the model, you must select, from the main menu, Display > User Properties.



In the user properties window, you have to choose:
  • Concept: select the project concept for which you want to define a property.

  • Name: identify the property by giving it a name.

  • Alias: additional identifier for the property.

  • Value type: specify the type of value that the user will enter for your property. For a list of the types of values (see table next page).

  • Description: enter a textual description (optional).


NOTE: When you change the Value Type, the values already entered will be lost.
