Notations


Gane-Sarson is the default notation when you create a new process model. You can change this notation for another one before, during or after creating the model.

To change the notation:
  1. Right-click in the diagram background.

  2. Choose in Notation in the pop-up menu.

  3. Select the new notation in the list. The diagram display will change immediately.

To avoid doing this operation on each new diagram, you can choose a default notation for the project. In the Project pop-up menu, you will find Default Notation (Process modeling).

If you want to use a customised notation set, refer to Notations.
