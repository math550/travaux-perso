Renumbering


The Renumber command lets you renumber all graphic representations of the same kind on a diagram. It calls up a window providing a Selective Renumbering option.


Selective renumbering

Choosing the parameters:
  1. In the Tools menu, choose Renumbering Parameters.

  2. Indicate the first number to be used for the selective renumbering in the Renumber from box.

  3. Specify the renumbering interval in the Increment by box.

  4. Click Close.



To proceed with a selective renumbering:
  1. Double-click the Renumbering tool from the Process creation toolbar.

  2. Click each graphic representation separately to renumber.

  3. Double-click the Select tool.
