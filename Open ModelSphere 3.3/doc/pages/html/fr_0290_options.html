Modification des options


Une autre façon de personnaliser l’environnement de travail est de modifier les options d'application. Dans le menu Outils, choisissez Options.

Les options sont classées en sept champs :
  • Général
  • Répertoire
  • Affichage
  • Diagramme
  • Espace de travail
  • Barres d'outils
  • Avancées

Les options Générales de l'application

  • Niveau d'annuler / rétablir: ce paramètre défini combien d'entrées sont conservées dans l'historique des commandes, afin de permettre l'annulation ou le rétablisement d'une annulation sur une action faite durant le travail.

  • Projets récemment ouverts: ce paramètre défini le nombre de fichier récent conservé en mémoire dans le menu Fichier / Rouvrir.

Les options de répertoires

Vous pouvez choisir les répertoires où seront générés certains fichiers, au lieu de garder les configurations par défaut. Il est donc possible de personnaliser les choix de répertoires suivants :
  • Répertoire de travail par défaut
  • Répertoire de génération des fichiers DDL
  • Répertoire de génération Java
  • Répertoire de génération HTML.
Les options d'affichage

L'interface utilisateur graphique (GUI)

Si vous trouvez que la fenêtre Explorateur et le panneau de conception prennent trop d'espace ou sont sur le mauvais côté de votre écran, vous pouvez les modifier. Des changements peuvent être apportés à la disposition de l'explorateur et du panneau de conception (soit gauche-droite, gauche-en bas ou gauche-gauche). Il est également possible d’afficher (ou cacher) les en-têtes des composantes d’interface usager.

Pour accéder à cette option, dans le menu principal choisissez Outils > Options > Affichage > Interface utilisateur graphique.


Apparence et convivialité

Il y a plusieurs configurations différentes offertes afin de modifier l’apparence et la convivialité de votre Open ModelSphere. Pour accéder à cette option, dans le menu principal choisissez Outils > Options > Affichage > Apparence et convivialité.


Grandeur par défaut des fenêtres

Vous pouvez personnaliser la taille de la fenêtre du diagramme et des fenêtres de propriétés. Choisissez Outils > Options > Affichage > Grandeur par défaut de la fenêtre.

Vous pouvez redimensionner les fenêtres : petit, moyen, grand, 3/4 d’espace disponible ou espace disponible maximal. La configuration par défaut est 3/4 d’espace disponible.


Les info-bulles de l'explorateur

Cette section permet de sélectionner les propriétés que l'on voudra voir s'afficher dans les info-bulles de l'explorateur, permettant un survol rapide des valeurs d'un objet sans avoir à faire de sélection. Choisissez Outils > Options > Affichage > Info-bulles de l'explorateur.


Langue d'affichage

Permet de choisir la langue de l'application. Choisissez Outils > Options > Affichage > Langue d'affichage.

Il est à noter que le changement se fera uniquement au redémarrage de l'application.


Les options du diagramme

Création automatique

Une autre façon de personnaliser l’espace de travail est de modifier les options du diagramme. D'abord, vous pouvez activer ou désactiver la création automatique du diagramme d'un modèle. Choisissez Outils > Options > Diagramme > Création automatique.
Vous pouvez rendre la création du diagramme automatique lors de la création d’un :
  • modèle de données
  • modèle de domaines
  • modèle d’items communs
  • paquetage Java
  • contexte d’un modèle de flux de données.
Si la fonction Création automatique est désactivée, les diagrammes sont disponibles manuellement.

La dernière option, est pour la création automatique ou non du cadre de contexte dans les diagrammes de processus.


Le cartouche

Vous pouvez aussi personnaliser le cartouche en changeant le logo fourni par défaut par le logo de votre compagnie. Pour changer le logo, choisissez Outils > Options > Diagramme > Cartouche. De là, vous devez Parcourir les répertoires jusqu’au fichier contenant l'image du logo de votre compagnie (.jpg ou .gif). Choisissez-le et cliquez Ouvrir.
Par contre, si vous préférez ne pas avoir de logo sur le cartouche, désactivez-le en décochant la case: Utiliser le logo dans le cartouche.
Pour retrouver le logo par défaut, cliquez sur Défaut.

Après toute opération, cliquez sur Appliquer afin de valider votre choix.


Zoom et loupe

La dernière option du diagramme concerne le niveau du zoom de la loupe et de la fenêtre du diagramme. Vous pouvez configurer le niveau d’agrandissement de la loupe au pourcentage de votre choix, de 60 à 300 %. Il est également possible de choisir le niveau du zoom par défaut de la fenêtre du diagramme au pourcentage de votre choix, de 50 à 150 %.

Faites simplement glisser le curseur au pourcentage désiré. Cliquez sur Appliquer afin de valider votre choix.

Les options de barres d'outils

Il est possible de configurer quels outils vous désirez voir apparaître sur les différentes barres d'outils.

Pour faire un changement, cochez ou décochez la case appropriée.
Vous pouvez contrôler l'affichage des barres d'outils en passant par le menu Affichage / Barres d'outils, et déterminer quels outils vous désirez voir s'afficher à l'intérieurs des barres d'outils par le menu Outils / Options > Barres d'outils.
