Ajouter une ressource


Une ressource peut être liée à un processus. Si tel est le cas, la représentation graphique de la ressource est affichée à l’intérieur du processus. On affiche le nom et/ou le temps et/ou le coût de chaque ressource du modèle qui est liée à ce même processus.

Pour ajouter une ressource :
  1. De l’explorateur, cliquez le modèle de processus avec le bouton droit.

  2. Dans le menu contextuel, choisissez Ajouter > Ressource.

Pour assigner une ressource à un processus :
  1. De l'explorateur, choisir une ressource avec la souris.


  2. Tenir le bouton de la souris et glissez la ressource sur un processus se trouvant dans le diagramme ou l'explorateur.


  3. Déposer la ressource en relâchant le bouton de la souris quand le processus devient sélectionné et que l'icone sous le curseur de la souris devient '+'.
OU
  1. Dans le menu pricipal, choisissez Affichage > Barre d'outils > Ressources, afin de faire apparaître la palette des ressources.

  2. Dans la palette des ressources, choisissez le ou les ressources que vous voulez assigner. Choisisser l'une des trois flèches selon que vous désirez ajouter, rendre identique ou supprimer votre sélection au processus visé.

  3. Puis cliquez simplement sur le processus visé.

La fenêtre de listes des Ressources contient un menu qui donne accès aux pages associées à chaque ressource. Après avoir sélectionné le nom d’une ressource, vous pouvez accéder à ces pages en choisissant le titre de la page désirée dans le menu.


Les propriétés

L’onglet Conception de la fenêtre de propriétés fournit l’information suivante :
  • le nom de la ressource

  • l’identifiant numérique qui lui est attribué automatiquement lors de sa création

  • l’alias que vous pouvez utiliser pour afficher un autre nom

  • le nom d’une catégorie si vous voulez regrouper des ressources

  • le nom physique

  • l’utilisateur si, par exemple, la ressource est une machine

  • le coût.

La fenêtre renseigne aussi sur la charge de travail de la ressource représentée par la somme de tous les taux d’utilisation de la ressource à l’intérieur de tous les processus.
