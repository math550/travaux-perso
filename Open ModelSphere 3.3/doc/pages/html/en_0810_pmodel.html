Building a process model


It is possible to create a process model from the main menu, from the explorer window and from the Edit toolbar.
  • From the main menu: choose Edit > Add > Business Process Model.

  • From the explorer window: right-click project. Then, from the pop-up menu, choose Add > Business Process Model.

  • From the Edit toolbar: click the Add tool and choose Business Process Model.

At the model level there is no diagram yet. Since process models can contain more than one context, diagrams are added at the context level.
