Ajouter un Système

Pour ajouter un système à un modèle UML, vous devez d'abord sélectionner le modèle UML dans l'explorateur...
  • À partir du menu principal: choisissez Édition > Ajouter > Système
  • Depuis l'explorateur: cliquez-droit sur le modèle UML et depuis le menu contextuel, choisissez Édition > Ajouter > Système.
  • Par la barre d'outils Édition : cliquez sur l'Outil Ajouter et choisissez Système.



Vous pouvez changer le nom «Système» automatiquement attribué par le logiciel en utilisant la fenêtre de description du système racine. Un changement de nom se répercutera immédiatement dans la barre de titre de sa fenêtre graphique.
