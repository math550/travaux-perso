Grouping Diagrams


Two or more diagrams can be grouped together in order to form only one diagram. The format will be kept for each diagram and the notation can be chosen by the user if different notation were assigned to the diagrams.

To group diagrams:

  1. Select the Diagrams to be grouped using the Ctrl key.
  2. Right-click on the Diagrams.
  3. Select Group Diagrams.


Or, you can do the following:

  1. Select the Diagrams to be grouped using the Ctrl key.
  2. Tools > Group Diagrams.

NOTE: It is also possible to invoke the Group Diagram option after selecting a Model. This will group all diagrams within the selected model.
