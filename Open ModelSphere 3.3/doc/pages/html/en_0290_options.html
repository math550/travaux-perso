Modifying the options


A few options of the work area can be modified or customized. In the Tools menu, choose Options.

The options are categorized as follows:
  • General
  • Directories
  • Display
  • Diagram
  • Workspace
  • Toolbars
  • Advanced

General application options

  • Undo / Redo Level: This parameter defines how many historic entries are kept for the Undo / Redo operations during the work.

  • Recent Files Items: This parameter defines how many file' paths are kept in the menu File / Reopen.

Directory options

You can choose directories instead of having default settings. Customization is possible for the following:
  • Default working directory
  • DDL generation directory
  • Java forward directory
  • HTML generation directory
Display options

Graphical User Interface (GUI)

If you think that the explorer window and the design panel take too much space or are on the wrong side of your screen, here is the place to change them. Changes can be made to the explorer and design panel layout (Left-Right, Left-Bottom or Left-Left) and GUI components headers can be shown or hidden.

To access this option, in the main menu, choose Tools > Options > Display > Graphical User Interface.


Look & Feel

There are eight different Look and Feel settings available allowing you to customize the appearance of your copy of Open ModelSphere. To use this option, in the main menu, select Tools > Options > Display > Look and feel.


Windows Default Size

You can customize the size of both the diagram window and the properties window. Choose Tools > Options > Display > Windows Default Size.

You can size the windows to: small, medium, large, 3/4 available space or maximum available space.
Default setting is 3/4 available space.


Explorer Tool Tips

This section lets you select the properties that you want to be displayed in the tool tips of the explorer. You have the opportunity to see rapidly, without selection, some values of the object. Choose Tools > Options > Display > Exporer Tool tips.


Display Language

You can choose the language used in the application. Choose Tools > Options > Display > Display Language.

NOTE: The changes are available only after restarting the application.


Diagram options

Automatic Creation

Another way to customize the working area is to use the diagrams options. You can enable or disable the automatic creation of a model’s diagram. Choose Tools > Options > Diagram > Automatic Creation.
You can apply automatic creation on:
  • data models
  • domain models
  • common item models
  • Java class models
  • Context of business process models
If Automatic creation is disabled, diagrams are only available through manual operations.

You can opt to create (or not) the context frame in process diagrams.


Stamp

You can also customize the stamp by replacing the default logo with your prefered logo.

To replace the logo, choose Tools > Options > Diagram > Stamp.

From there, you must Browse to the folder containing the picture of your prefered logo, select it and click Open. (.jpg or .gif) If, instead, you prefer not to have any logo on the stamp, disable Use a Stamp Logo by unchecking the box.

To return to the default logo, click Default.

After any operation, click Apply to validate your selection.


Zoom and Magnifier

The last diagram option deals with the zoom level of both the magnifier window and the diagram window. It lets you set the magnifier zoom level to any percentage, from 60 to 300%. It also lets you choose the default zoom level for the diagram window to any percentage, from 50 to 150%.

Simply slide the cursor to the desired percentage. Click Apply to validate your selection.

Toolbar options

If you do not like to have more than one line of tools, you might want to use the Toolbar options. Not only can choose which toolbar will be display or not, you can also choose which buttons of the different toolbars will be displayed and which buttons will not.

To make a change, check or uncheck the appropriate box. Of course, any of these changes are not irreversible! This simply sets customized default settings. To modify only a few things, do it manually, one by one. If you need to readjust most or all customizable default settings, use Tools > Options menu.
