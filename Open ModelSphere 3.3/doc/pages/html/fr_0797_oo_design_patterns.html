Application des patrons de conception orienté-objet
 

Il est possible d'appliquer un patron de conception sur une classe existante. Par exemple, si vous voulez appliquer le patron de conception  Singleton design pattern sur une classe nommée PrinterManager:

  1. Dans le diagramme de classe, créez une classe et nommez-la PrinterManager
  2. Selectionnez la classe en utilisant le bouton droit de la souris, et dans le menu déroulant, choisissez Appliquer le patron de conception->Singleton
Votre classe est renommée PrinterManagerSingleton, et les éléments du patron de conception apparaîtront en tant que champs et méthodes de la classe. Une note apparaît aussi pour décrire comment utiliser la classe à partir de code externe.

Les patrons de conception sont regroupés en trois catégories : création, structure et comportement. Les patrons de conceptions suivants sont présentement implantés :
