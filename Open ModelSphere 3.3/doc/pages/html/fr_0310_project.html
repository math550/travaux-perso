Le concept de projet


Selon la hiérarchie de l’explorateur, le projet est le conteneur suprême de toute l’information de modélisation et de conception. Toute l'information sera contenue dans le projet. Un objet ne peut pas faire référence à un objet externe. On ne permet aucune référence à l'extérieur du projet. Autrement dit, on ne permet aucune liaison sémantique inter-projet.


Les composantes d’un projet

Un projet inclut une panoplie d'objets sémantiques ou graphiques. Toutes ces composantes peuvent être classées dans deux catégories : prédéfinies ou définies par l'utilisateur.

Pour voir les composantes d'un projet, vous devez d'abord créer un nouveau projet. Dans le menu principal, choisissez Fichier > Nouveau.
